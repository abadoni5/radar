## Radar ToolBox

The radar engineering project website aims to provide a comprehensive educational resource on
radar systems. The website is structured into three key sections: Learn Radar, Radars Parameters,
and a Radar Calculator. This report will explain the purpose and interconnectedness of these
three components, highlighting how they work together to offer a well-rounded learning
experience for users.
